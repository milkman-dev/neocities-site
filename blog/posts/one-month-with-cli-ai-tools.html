one month with CLI AI tools

08 February 2026

i've been using AI tools in the terminal for the past month and i want to write about it while it still feels new, before i get used to it and forget what changed

the thing that's hard to explain to someone who hasn't tried it is that it doesn't feel like using a tool, it feels like working with something, the conversation stays in context, you can point it at a file and it reads it, ask it to make a change and it does, ask it why it made that choice and it explains and you can push back, and it responds to that

my workflow changed more in this one month than it has in the past year, not because i'm doing less but because the friction of certain things dropped to nearly zero, looking up syntax, finding the bug in a function, understanding a codebase i didn't write, all of that got faster in a way that compounds

the part that surprised me most is that i'm actually learning more, i expected to use it as a shortcut and instead i end up in conversations about why something works a certain way, it corrects me when i'm doing something wrong and explains it, which is exactly what learning alone was missing

i don't know what this looks like in a year, i don't think anyone does, but the potential feels like something i haven't felt about a tool since i first understood what a database could do, that feeling of something opening up rather than closing

this is going to change things a lot, i'm certain of that now
